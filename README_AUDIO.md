# 🎧 听力音频功能说明

## ⚠️ 当前状态

由于技术限制，听力音频功能需要额外配置才能正常使用。

## 🔧 问题说明

- **CDN域名：** 七牛云测试域名 `t6r1sg3dy.hd-bkt.clouddn.com`
- **协议：** 仅支持 HTTP（不支持HTTPS）
- **网站协议：** HTTPS（Cloudflare Pages）
- **冲突：** 浏览器阻止HTTPS网站加载HTTP资源（混合内容阻止）

## 📝 临时解决方案（仅测试用）

### 在Chrome/Edge浏览器中允许不安全内容：

1. 打开听力测试页面
2. 点击地址栏右侧的 🔒 锁图标
3. 选择"网站设置" 
4. 找到"不安全内容"选项
5. 改为"允许"
6. 刷新页面

### 在Firefox浏览器中：

1. 打开听力测试页面
2. 点击地址栏左侧的 🛡️ 盾牌图标
3. 选择"关闭此站点的保护"
4. 刷新页面

## ✅ 长期解决方案

为了在生产环境中正常使用，需要：

### 选项1：配置支持HTTPS的CDN
1. 购买/绑定自定义域名到七牛云
2. 配置HTTPS证书
3. 更新代码中的CDN域名

### 选项2：使用Cloudflare R2
1. 上传音频文件到Cloudflare R2存储
2. 配置公开访问
3. 更新音频链接

### 选项3：使用其他HTTPS CDN
- 阿里云OSS + 自定义域名
- 腾讯云COS + 自定义域名  
- AWS S3 + CloudFront

- **格式：** MP3
- **总计：** 53套，约320MB
