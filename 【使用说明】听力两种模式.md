## 🎓 考试模式详解

### 使用场景
- ✅ 考前1-2周模拟
- ✅ 检验复习效果
- ✅ 适应考试节奏
- ✅ 测试真实水平

### 操作流程

```
1. 选择试卷
   ↓
2. 选择"考试模式"
   ↓
3. 点击"开始考试"
   ↓
4. 音频自动播放（不可暂停）
   ↓
5. 边听边答题（25分钟）
   ↓
6. 时间到或完成后提交
   ↓
7. 查看成绩和解析
```

### 特性说明

#### 1. 严格限时
- ⏰ **25分钟倒计时**
- ⏱️ 时间到自动提交
- 🚫 不可延长时间

#### 2. 音频控制
- ▶️ 自动播放完整音频
- 🚫 不可暂停
- 🚫 不可快进/后退
- 🚫 不可重听
- ✅ 只能播放一遍（符合真实考试）

#### 3. 答题规则
- 📝 必须在音频播放过程中或播放后立即答题
- 🚫 不可查看听力原文
- 🚫 不可实时查看解析
- ✅ 可以先不选，最后统一提交

#### 4. 提交与评分
- 📊 一次性提交所有答案
- ✅ 立即显示成绩
- 📈 显示正确率、用时、排名
- 📖 提交后可查看详细解析

## 📚 练习模式详解

### 使用场景
- ✅ 日常练习
- ✅ 专项突破
- ✅ 精听训练
- ✅ 查漏补缺

### 操作流程

```
1. 选择试卷
   ↓
2. 选择"练习模式"
   ↓
3. 选择Section或单独题目
   ↓
4. 播放音频（可暂停）
   ↓
5. 答题（可实时看解析）
   ↓
6. 不满意可重做
   ↓
7. 查看统计和建议
```

### 特性说明

#### 1. 灵活时间
- ⏰ **不限时**
- 🕐 想练多久练多久
- ✅ 可以分段练习
- ✅ 可以中途暂停

#### 2. 音频控制
- ▶️ 可播放/暂停
- ⏮️ 可后退10秒
- ⏭️ 可前进10秒
- 🔁 可无限重听
- 🎚️ 可调节播放速度（0.5x - 2x）

#### 3. 辅助功能
- 📖 **可查看听力原文**（中英对照）
- 💡 **实时查看解析**（答题前后都可以）
- 🔍 **重点词汇标注**
- 📌 **错题自动收藏**

#### 4. 答题自由
- ✅ 可多次作答
- ✅ 可单题练习
- ✅ 可跳过题目
- ✅ 不计入正式成绩

### 辅助材料示例

```markdown
【听力原文】
M: Hi Sarah, have you finished your research proposal?
W: Not yet. I'm struggling with the methodology section.
M: What's the problem?
W: I'm not sure which approach would be most appropriate...

【重点词汇】
- research proposal  研究计划
- methodology  方法论
- appropriate  合适的

【解题技巧】
本题考查主旨大意，需要把握对话的核心内容。
关键句在开头"struggling with the methodology section"。
```

## 🎯 使用建议

### 学习路径推荐

#### 阶段1：基础练习（第1-2周）
```
练习模式 → 单个Section → 反复听
↓
重点：
- 熟悉题型
- 积累词汇
- 理解原文
```

#### 阶段2：专项提升（第3-4周）
```
练习模式 → 完整试卷 → 对照原文
↓
重点：
- 提高速度
- 掌握技巧
- 分析错题
```

#### 阶段3：模拟冲刺（考前1周）
```
考试模式 → 完整试卷 → 严格限时
↓
重点：
- 适应节奏
- 测试水平
- 调整心态
```

### 每日练习计划

#### 工作日（30-45分钟）
```
练习模式
├─ Section A: 1篇对话（10分钟）
├─ Section B: 1篇短文（10分钟）
└─ 复习错题（10分钟）
```

#### 周末（60-90分钟）
```
考试模式
├─ 完整模拟考（25分钟）
├─ 对答案分析（20分钟）
└─ 精听难点（15分钟）
```

## 💡 使用技巧

### 考试模式技巧

1. **预读题目**
   - 播放前快速浏览所有题目
   - 划出关键词
   - 预测可能的答案

2. **边听边记**
   - 记录关键词、数字、人名
   - 标注选项（排除法）
   - 不确定的先跳过

3. **时间分配**
   - Section A: 5-7分钟
   - Section B: 7-9分钟
   - Section C: 10-12分钟

4. **答题策略**
   - 先易后难
   - 不纠结单题
   - 确保全部答完

### 练习模式技巧

1. **精听训练**
   - 第一遍：听大意
   - 第二遍：听细节
   - 第三遍：对照原文

2. **词汇积累**
   - 记录生词
   - 整理短语
   - 复习巩固

3. **错题分析**
   - 为什么错？
   - 正确思路是什么？
   - 类似题怎么做？

4. **速度训练**
   - 从0.8x开始
   - 逐渐提升到1x
   - 最后挑战1.2x

## ⚠️ 注意事项

### 考试模式注意

1. **设备准备**
   - 确保网络稳定
   - 确保音频正常
   - 选择安静环境

2. **考前确认**
   - 检查是否选对试卷
   - 确认是"考试模式"
   - 准备好纸笔

3. **考试中**
   - 不要刷新页面
   - 不要切换标签
   - 不要中途退出

### 练习模式建议

1. **不要过度依赖**
   - 不要频繁暂停
   - 不要立即看原文
   - 先尝试理解

2. **合理使用辅助**
   - 第一遍不看原文
   - 实在不懂再查看
   - 看完要回听

3. **定期自测**
   - 每周至少1次考试模式
   - 检验练习效果
   - 调整学习计划
