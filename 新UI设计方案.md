# WeHUSTER风格UI重设计方案

## 🎯 设计目标
将当前网站改造成类似 wehuster.com/cet4 的极简风格，保留所有现有功能。

## 🎨 核心设计元素

### 1. 整体风格
- **配色**：白色背景 + 深色文字 + 蓝色链接
- **布局**：单栏布局，最大宽度约1200px居中
- **字体**：系统默认字体，清晰易读
- **间距**：宽松的留白，提升阅读体验

### 2. 导航栏
- 简洁的顶部导航
- Logo + 主要链接
- 白色背景，细边框分隔

### 3. 真题列表页面（CET4/CET6）
- **页面标题**：大号字体，简洁明了
- **简介文字**：灰色小字，说明考试信息
- **年份分组**：使用 h3 标题清晰分组
- **真题卡片**：列表式展示
  - 每套真题一行
  - 真题标题 + 答案链接并排
  - 悬停效果：背景色变化
  - "含听力"标签突出显示
- **批量下载**：顶部添加批量下载按钮

### 4. 真题详情页
- 保持现有功能：
  - 试题展示
  - 听力音频播放
  - PDF下载/打印
- 采用简洁的卡片样式

### 5. 首页
- Hero区域简化
- 突出核心功能：四级/六级真题入口
- 简洁的特色功能介绍
