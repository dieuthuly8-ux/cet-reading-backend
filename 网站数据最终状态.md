## 🛠️ 技术实现

### 1. PDF解析优化

```javascript
// exam-loader.js - resolvePdfUrlUnified 函数
// 优先级：papers.json > exam-contents.json > 路径推导
```

**解析流程**：
1. 从 `papers.json` 读取PDF路径（支持四级和六级）
2. 失败则从 `exam-contents.json` 读取
3. 最后使用路径推导规则

### 2. 数据配置格式

```json
{
  "id": "cet6-2025-06-set1",
  "title": "2025年6月 六级真题 (第一套)",
  "year": 2025,
  "month": "06",
  "pdf": "assets/cet6-pdf/CET-6 2025.06 第1套.pdf"
}
```
