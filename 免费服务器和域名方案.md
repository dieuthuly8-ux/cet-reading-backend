```bash
# 1. 安装Vercel CLI
npm install -g vercel

# 2. 登录Vercel
vercel login

# 3. 在项目目录下部署
cd E:\cursor\作业

**或使用网页方式：**

4. 点击Deploy

```

```

**示例：**

https://cet4-cet6.vercel.app
```

### 优点

## 📝 方案B：Vercel + 免费.eu.org域名（专业）

### 第1步：注册免费域名

**EU.org提供免费二级域名**

1. **访问注册页面**
   - https://nic.eu.org/

2. **注册账号**
   - 填写基本信息
   - 验证邮箱

3. **申请域名**
   - 登录后点击"New Domain"
   - 输入你想要的域名，如：`cet-learning`
   - 完整域名将是：`cet-learning.eu.org`
   - 填写DNS信息（下一步提供）

4. **等待审核**
   - 通常需要1-7天审核
   - 审核通过后会收到邮件

### 第2步：部署到Vercel

1. 按照方案A部署网站
2. 获得Vercel域名：`xxx.vercel.app`

### 第3步：绑定自定义域名

**在Vercel控制台：**

1. 进入项目设置
2. 点击 **Domains**
3. 添加你的域名：`cet-learning.eu.org`
4. Vercel会显示DNS记录要求

**在EU.org控制台：**

1. 登录EU.org账号
2. 管理你的域名
3. 添加DNS记录（根据Vercel要求）：
   ```
   类型: CNAME
   名称: @
   值: cet-learning.vercel.app
   ```

4. 等待DNS生效（最多48小时，通常几分钟）

### 完成！

你的网站现在可以通过以下方式访问：
- ✅ `https://cet-learning.eu.org`（你的自定义域名）
- ✅ `https://cet-learning.vercel.app`（备用域名）

## 📝 方案C：Cloudflare Pages（中国速度更快）

### 为什么选择Cloudflare Pages？

✅ **中国速度快** - Cloudflare在中国有CDN节点  
✅ **完全免费** - 无限流量和构建  
✅ **自带域名** - `xxx.pages.dev`  

### 部署步骤

**第1步：注册Cloudflare**
1. 访问 https://pages.cloudflare.com/
2. 注册账号（完全免费）

**第2步：连接GitHub**
1. 点击 **Create a project**
2. 连接你的GitHub账号
3. 选择你的项目仓库

**第3步：配置项目**
- **Build command**: 留空
- **Build output directory**: `/`
- **Root directory**: `/`

**第4步：部署**
- 点击 **Save and Deploy**
- 等待3-5分钟

**第5步：获得域名**
```
https://cet-learning.pages.dev
```

## 📝 方案D：GitHub Pages（简单但速度一般）

### 优点
- ✅ 最简单的方案
- ✅ 完全免费
- ✅ GitHub官方服务

### 缺点
- ⚠️ 中国访问较慢
- ⚠️ 域名固定为：`username.github.io/repo-name`

### 部署步骤

**第1步：创建GitHub仓库**
- 仓库名：`cet-learning`
- 设置为Public

**第2步：推送代码**
```bash
cd E:\cursor\作业
git init
git add .
git commit -m "部署到GitHub Pages"
git remote add origin https://github.com/你的用户名/cet-learning.git
git push -u origin main
```

**第3步：启用GitHub Pages**
1. 进入仓库设置
2. 找到 **Pages** 选项
3. Source选择 `main` 分支
4. 点击 **Save**

**第4步：访问网站**
```
https://你的用户名.github.io/cet-learning/
```
