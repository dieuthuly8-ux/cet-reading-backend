# 🚀 网站部署操作指南

## 📝 前言

本指南将帮助你把**四六级学习站**部署到互联网，让所有人都可以访问。

**部署平台：Vercel**（完全免费，无需信用卡）

---

## 🎯 部署步骤（15分钟完成）

### 第1步：准备工作

#### 1.1 创建GitHub账号（如已有请跳过）
- 访问：https://github.com
- 点击 **Sign up** 注册账号
- 验证邮箱

#### 1.2 将项目上传到GitHub

**方式一：使用Git命令行**
```bash
# 在项目目录下执行
cd E:\cursor\作业

# 初始化Git仓库（如果还没有初始化）
git init

# 添加所有文件
git add .

# 提交
git commit -m "准备部署到Vercel"

# 在GitHub上创建新仓库后，连接远程仓库
git remote add origin https://github.com/你的用户名/cet-learning.git

# 推送到GitHub
git push -u origin main
```

**方式二：使用GitHub Desktop（图形界面）**
1. 下载安装 [GitHub Desktop](https://desktop.github.com/)
2. 登录GitHub账号
3. 点击 **File** → **Add Local Repository**
4. 选择项目文件夹 `E:\cursor\作业`
5. 点击 **Publish repository** 发布到GitHub

---

### 第2步：部署到Vercel

#### 2.1 注册Vercel账号
1. 访问：https://vercel.com
2. 点击 **Sign Up**
3. 选择 **Continue with GitHub**（使用GitHub账号登录）
4. 授权Vercel访问你的GitHub仓库

#### 2.2 导入项目
1. 进入Vercel控制台
2. 点击 **Add New...** → **Project**
3. 找到你刚才上传的仓库（例如：`cet-learning`）
4. 点击 **Import**

#### 2.3 配置项目（重要！）

在配置页面：

**Framework Preset**: 选择 `Other`（或留空）

**Root Directory**: 留空（使用项目根目录）

**Build and Output Settings**:
- Build Command: 留空
- Output Directory: 留空
- Install Command: 留空

**Environment Variables（环境变量）**:

如果你需要使用**翻译功能**，需要配置百度翻译API：

| 变量名 | 值 | 说明 |
|--------|-----|------|
| `BAIDU_APPID` | 你的百度APP ID | 从百度翻译开放平台获取 |
| `BAIDU_SECRET` | 你的百度密钥 | 从百度翻译开放平台获取 |

**如何获取百度翻译API密钥？**
1. 访问：https://fanyi-api.baidu.com/
2. 注册/登录百度账号
3. 点击 **管理控制台**
4. 开通 **通用翻译API**（标准版免费）
5. 创建应用，获得 APP ID 和 密钥

**注意**：如果暂时不需要翻译功能，可以跳过此步骤，其他功能不受影响。

#### 2.4 开始部署
1. 检查配置无误
2. 点击 **Deploy** 按钮
3. 等待3-5分钟（首次部署需要时间）

#### 2.5 部署成功
部署完成后，Vercel会自动生成一个网址，例如：
```
https://cet-learning.vercel.app
```

🎉 **恭喜！你的网站已经上线了！**

## 🌐 访问你的网站

部署成功后，你可以通过以下方式访问：

### 默认域名
```
https://你的项目名.vercel.app
```

### 自定义域名（可选）
1. 在Vercel项目设置中点击 **Domains**
2. 添加你自己的域名（如果你有的话）
3. 按照提示配置DNS记录

## 🔄 如何更新网站

```bash
git push origin main
```

等待2-3分钟，网站就会自动更新！

## 🛠️ 常见问题

### Q1: 部署失败怎么办？
**A**: 检查以下几点：
- 确保项目已正确上传到GitHub
- 查看Vercel部署日志，找到具体错误信息
- 确保`.gitignore`文件正确配置（不要忽略重要文件）

### Q2: 翻译功能不工作？
**A**: 检查：
- Vercel环境变量是否正确配置
- 百度API密钥是否有效
- 百度翻译API是否有剩余额度（免费版每月5万字符）

### Q3: 网站加载慢？
**A**: 
- Vercel在中国访问可能稍慢，这是正常现象
- 可以考虑使用CDN加速
- 或使用国内平台部署（如：阿里云、腾讯云）

### Q4: 如何删除部署的网站？
**A**: 
- 进入Vercel控制台
- 选择项目 → Settings → Advanced
- 点击 **Delete Project**

### Q5: 需要付费吗？
**A**: 
- **不需要！** Vercel免费版完全够用
- 免费版限制：每月100GB带宽、无限项目
- 你的网站流量不会超过这个限制
