# ⚡ 快速部署 - 3步搞定

## 🎯 只需3步，让你的网站上线！

---

## 📋 第1步：上传到GitHub（5分钟）

### 方法A：使用GitHub Desktop（推荐，最简单）

1. **下载GitHub Desktop**
   - 访问：https://desktop.github.com/
   - 下载并安装

2. **登录GitHub**
   - 打开GitHub Desktop
   - 点击 **Sign in to GitHub.com**
   - 输入你的GitHub账号（没有就先注册一个）

3. **上传项目**
   - 点击 **File** → **Add Local Repository**
   - 选择文件夹：`E:\cursor\作业`
   - 点击 **Publish repository**
   - 取消勾选 **Keep this code private**（公开仓库）
   - 点击 **Publish Repository**

✅ **完成！** 你的代码已经在GitHub上了！

### 方法B：使用命令行

```bash
cd E:\cursor\作业
git init
git add .
git commit -m "首次提交"
git branch -M main
git remote add origin https://github.com/你的用户名/cet-learning.git
git push -u origin main
```

---

## 🚀 第2步：部署到Vercel（5分钟）

1. **访问Vercel**
   - 打开：https://vercel.com
   - 点击 **Sign Up**
   - 选择 **Continue with GitHub**

2. **导入项目**
   - 点击 **Add New...** → **Project**
   - 找到刚才上传的仓库
   - 点击 **Import**

3. **配置（可选）**
   
   **如果需要翻译功能**，添加环境变量：
   - `BAIDU_APPID` = 你的百度APP ID
   - `BAIDU_SECRET` = 你的百度密钥
   
   **不需要翻译功能？** 直接跳过此步！

4. **点击Deploy**
   - 等待3-5分钟
   - 完成！

---

## 🌐 第3步：访问你的网站

部署完成后，Vercel会给你一个网址：

```
https://你的项目名.vercel.app
```

🎉 **恭喜！网站已上线，所有人都能访问了！**
