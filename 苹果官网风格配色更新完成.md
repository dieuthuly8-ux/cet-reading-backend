# 🍎 苹果官网风格配色更新完成

## 🎨 核心改变

已完全抛弃原有的莫兰迪配色，采用**苹果官网的配色和UI风格**！

## 💎 全新配色系统

### 背景色
```css
--color-bg: #ffffff;           /* 纯白背景 */
--color-bg-secondary: #f5f5f7; /* 浅灰背景 */
```
**对比**：
- ❌ 原来：浅绿色 #E5EFC1
- ✅ 现在：纯白色 #ffffff

### 主色调
```css
--color-primary: #0071e3;      /* 苹果蓝 */
--color-primary-hover: #0077ED; /* 悬停更亮蓝 */
```
**对比**：
- ❌ 原来：珊瑚橙 #F98C53
- ✅ 现在：苹果蓝 #0071e3

### 文字色
```css
--color-text-primary: #1d1d1f;   /* 苹果黑 */
--color-text-secondary: #6e6e73; /* 苹果灰 */
```
**对比**：
- ❌ 原来：深绿 #6B7C68
- ✅ 现在：深灰黑 #1d1d1f

### 卡片色
```css
--color-card: #ffffff;         /* 纯白 */
--color-card-hover: #f5f5f7;   /* 浅灰 */
```
**对比**：
- ❌ 原来：米白 #FEFCFA
- ✅ 现在：纯白 #ffffff

## 🎯 视觉变化

### 1. 整体背景
```
原来：浅绿色（温暖莫兰迪风）
现在：纯白色/浅灰色（简洁现代风）✅
```

### 2. 主按钮
```
原来：珊瑚橙圆角按钮 + 阴影
现在：苹果蓝极圆角按钮 + 无阴影 ✅
```
- 圆角：980px（极圆角）
- 悬停：scale(1.02)
- 无阴影

### 3. 卡片样式
```
原来：米白背景 + 彩色阴影
现在：纯白背景 + 浅灰阴影 ✅
```
- 背景：#ffffff
- 边框：1px solid rgba(0,0,0,0.08)
- 阴影：0 2px 8px rgba(0,0,0,0.06)

### 4. 导航栏
```
原来：米白半透明 + 浅绿边框
现在：纯白半透明 + 极细边框 ✅
```
- 背景：rgba(255,255,255,0.72)
- 边框：inset 0 -1px 0 0 rgba(0,0,0,0.1)

### 5. 下拉菜单
```
原来：米白背景 + 珊瑚橙高亮
现在：白色背景 + 浅灰高亮 ✅
```

### 6. 搜索框
```
原来：米白背景 + 珊瑚橙聚焦
现在：白色背景 + 苹果蓝聚焦 ✅
```

## 🎯 参考的苹果设计

### Apple.com 配色
- 纯白背景
- 苹果蓝 #0071e3
- 深灰文字 #1d1d1f

### Apple 设计原则
- **简洁**：去除一切不必要的元素

- **对比**：高对比度确保可读性
- **一致**：统一的设计语言

### Apple UI 特征
- 极圆角按钮（980px）
- 细边框（1px）
