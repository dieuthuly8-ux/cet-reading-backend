# 🔧 清除浏览器缓存 - 详细步骤

## 问题现象
- 网站显示"共找到 0 套真题"
- 试卷列表为空
- 听力资源看不到

## 原因
浏览器缓存了旧版本的网页文件（HTML、CSS、JS）

---

## 解决方案

### ✅ 方法1：强制刷新（推荐，最快）

**在网站页面上按：**
- Windows: `Ctrl + Shift + R` 或 `Shift + F5`
- Mac: `Command + Shift + R`

**然后等待页面重新加载**

---

### ✅ 方法2：清除浏览器缓存（彻底）

#### Microsoft Edge:
1. 按 `Ctrl + Shift + Delete`
2. 选择"时间范围"：**最近 1 小时**
3. 勾选：
   - ☑ 浏览历史记录
   - ☑ 缓存的图像和文件
4. 点击"立即清除"
5. 刷新页面（F5）

#### Chrome:
1. 按 `Ctrl + Shift + Delete`
2. 选择"时间范围"：**最近 1 小时**
3. 勾选：
   - ☑ 浏览记录
   - ☑ 缓存的图片和文件
4. 点击"清除数据"
5. 刷新页面（F5）

---

### ✅ 方法3：无痕模式测试

**打开无痕窗口：**
- `Ctrl + Shift + N`（Chrome/Edge）

**然后访问：**
```
https://cet-study-2024.pages.dev/cet4.html
```

如果无痕模式下能看到试卷，说明确实是缓存问题。

## 如果还是不行

### 1. 检查网络连接
确保能访问：`https://cet-study-2024.pages.dev`

### 2. 查看浏览器控制台
1. 按 `F12` 打开开发者工具
2. 点击"Console"（控制台）标签
3. 刷新页面
4. 截图发送给我，看是否有错误信息

### 3. 检查文件加载
1. 按 `F12` 打开开发者工具
2. 点击"Network"（网络）标签
3. 刷新页面
4. 查找 `papers.json` 文件
5. 点击查看是否成功加载（状态码应该是 200）
