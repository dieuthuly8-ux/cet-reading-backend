### 步骤 2️⃣：创建表单（30秒）

### 步骤 3️⃣：获取 Form ID（30秒）

#### 3.1 表单创建成功后，你会看到这样的界面：

```

│ 四六级学习站联系表单               │

│                   这就是Form ID！  │

```

**Form ID 示例**：`xyzabc123`

#### 3.2 复制 Form ID

只需要最后的字符串部分，例如：`xyzabc123`

---

### 步骤 4️⃣：更新代码（30秒）

#### 4.1 打开 `index.html`

#### 4.2 搜索 `YOUR_FORM_ID`

找到第 288 行：

```html
<form id="contact-form" action="https://formspree.io/f/YOUR_FORM_ID" method="POST" ...>
```

#### 4.3 替换 Form ID

```html
<!-- ❌ 替换前 -->
<form id="contact-form" action="https://formspree.io/f/YOUR_FORM_ID" method="POST" ...>

<!-- ✅ 替换后（使用你的实际Form ID） -->
<form id="contact-form" action="https://formspree.io/f/xyzabc123" method="POST" ...>
```

#### 4.4 保存文件

**Ctrl + S** 保存

## 💡 常见问题

### Q1: 免费版够用吗？
**A:** 完全够用！
- 每月 50 封邮件
- 无限制的表单数量
- 除非网站特别火，否则不会超

### Q2: 会收到垃圾邮件吗？
**A:** 不会！
- Formspree 自带垃圾邮件过滤
- 有验证机制防止滥用

### Q3: 可以自定义邮件格式吗？
**A:** 可以！
- 在 Formspree 后台设置
- Settings → Email Notifications
- 自定义模板

### Q4: 如果超过50封怎么办？
**A:** 
- 可以升级到付费版（$10/月，1000封）
- 或者创建多个免费账号

### Q5: 表单数据会保存吗？
**A:** 会！
- Formspree 会保存所有提交记录
- 可以在后台查看和导出

## 🔍 故障排除

### ❌ 问题1：点击发送后显示"发送失败"

**可能原因**：
- Form ID 填错了
- 网络问题

**解决方法**：
1. 检查 Form ID 是否正确
2. 确保没有多余的空格
3. 检查网络连接
4. 按 F12 查看控制台错误

---

### ❌ 问题2：发送成功但收不到邮件

**可能原因**：
- 邮件进入垃圾箱
- 邮箱地址填错了

**解决方法**：
1. 检查垃圾邮件箱
2. 登录 Formspree 后台查看提交记录
3. 在 Formspree 设置中确认邮箱地址
4. 标记 Formspree 邮件为"非垃圾邮件"

---

### ❌ 问题3：表单提交后跳转到 Formspree 页面

**可能原因**：
- 代码中缺少 `e.preventDefault()`

**解决方法**：
- 不用担心，我已经在代码中处理了
- 如果还是跳转，检查 JavaScript 代码是否正确加载
